# MonitorControl.Web

ASP.NET Core host: JSON REST API (`/api/*`), **SSE** (`/api/events/monitor`), **WebSocket** (`/ws/monitor-watch`), Swagger UI, and static browser demo under `wwwroot/`.

Run: `dotnet run --project src/MonitorControl.Web --urls http://127.0.0.1:5080`

Documentation: [docs/guide/web-api-and-python-gateway.md](../../docs/guide/web-api-and-python-gateway.md)

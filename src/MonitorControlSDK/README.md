# Sony.MonitorControl

.NET 8 library: **SDAP** (UDP discovery on port **53862**), **SDCP** control on **TCP 53484** (`SdcpConnection` + `VmcClient`), and optional **UDP SDCP VMC** on **53484** for manual **Group / All** broadcast (`VmcUdpBroadcastClient`) per the PVM-740 programmer excerpt.
